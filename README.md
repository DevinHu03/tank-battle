## Game Controls

### Player 1
- **WASD** - Move tank
- **Space** - Shoot

### Player 2
- **Arrow Keys** - Move tank
- **Enter** - Shoot

### General Controls
- **F1** - Show controls info
- **ESC** - Pause/Back
- **R** - Restart
